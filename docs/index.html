⚡ Quick Setup

# Set your DigitalOcean API token
export DIGITALOCEAN_ACCESS_TOKEN="your_digitalocean_api_token"

# Build the MCP server
go build -o digitalocean-mcp-server

# Run the server
./digitalocean-mcp-server

# Or run directly with Go
go run main.go

🔧 Dependencies

go get github.com/digitalocean/godo
go get github.com/metoro-io/mcp-golang
go get golang.org/x/oauth2

📚 Architecture Overview

🎯 main.go

Entry point that initializes and starts the MCP server via server.Start() using stdio transport protocol.

🖥️ server/

MCP server configuration and tool registration. Contains server.go for initialization and tools.go for registering all 11 API tools.

🔌 client/

DigitalOcean API client wrapper using OAuth2 authentication. Handles secure communication with DO API endpoints.

⚡ handlers/

Business logic for each MCP tool including droplets, Kubernetes, and registry operations with standardized error handling.
