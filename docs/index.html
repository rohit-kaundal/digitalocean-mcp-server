DigitalOcean MCP Server

Comprehensive AI-powered Model Context Protocol server with 37 tools across 6 service categories. Complete DigitalOcean infrastructure management through intelligent automation.

🚀 Comprehensive Cloud Management (37 Tools)

🖥️

Droplet Management (7 tools)

Complete lifecycle management including create, list, resize, delete, and snapshot operations. Full CRUD with intelligent error handling and status monitoring.

💾

Volume Management (8 tools)

Block storage operations including attach/detach, resize, and snapshot functionality. Complete volume lifecycle management with automated backup capabilities.

📸

Snapshot Operations (6 tools)

Backup and restore functionality for both droplets and volumes. Automated snapshot management with filtering and cleanup operations.

🖼️

Image Management (6 tools)

Custom image operations including transfers, conversions, and metadata updates. Seamless image lifecycle management across regions.

🌐

Floating IP Management (6 tools)

Static IP allocation, assignment, and management. Dynamic IP assignment with automated failover and load balancing capabilities.

⚖️

Load Balancer Operations (9 tools)

Traffic distribution with full CRUD operations. Advanced forwarding rules, droplet pool management, and health monitoring.

☸️

Kubernetes Integration (4 tools)

Comprehensive cluster and node pool management. Deploy, scale, and monitor containerized applications with AI-driven insights.

📦

Container Registry (2 tools)

Access and manage DigitalOcean container registries. Pull images, manage repositories, and integrate with CI/CD pipelines.

🔐

Secure Authentication

OAuth2-based authentication with DigitalOcean API. Secure token management and encrypted communication channels for all operations.

⚡ Quick Setup

# Set your DigitalOcean API token
export DIGITALOCEAN_ACCESS_TOKEN="your_digitalocean_api_token"

# Build the MCP server
go build -o digitalocean-mcp-server

# Run the server
./digitalocean-mcp-server

# Or run directly with Go
go run main.go

🔧 Dependencies

go get github.com/digitalocean/godo
go get github.com/metoro-io/mcp-golang
go get golang.org/x/oauth2

📚 Architecture Overview

🎯 main.go

Entry point that initializes and starts the MCP server via server.Start() using stdio transport protocol.

🖥️ server/

MCP server configuration and tool registration. Contains server.go for initialization and tools.go for registering all 37 API tools across 6 service categories.

🔌 client/

DigitalOcean API client wrapper using OAuth2 authentication. Handles secure communication with DO API endpoints.

⚡ handlers/

Modular business logic with 8 specialized handler files: droplets, volumes, snapshots, images, floating IPs, load balancers, Kubernetes, and registry operations.
